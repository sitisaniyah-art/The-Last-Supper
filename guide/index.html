新手使用指南

欢迎来到"最后的晚餐"学习资源互助社区！这个页面将告诉你如何使用本网站的各个功能。

快速开始

浏览学习资源

前往学习资源页面，按学科、年级或类型筛选你需要的资料。点击"下载资源"按钮即可获取。

搜索特定资源

在资源页面的搜索框中输入关键词（如"高数期末"、"Python入门"），系统会实时过滤显示匹配的资源。

提出学习问题

前往疑问区，点击"我要提问"。你需要一个 GitHub 账号（免费注册），填写问题描述后提交即可。

注册 GitHub 账号

访问 GitHub

打开 github.com，点击"Sign up"注册账号。只需要邮箱即可，完全免费。

验证邮箱

GitHub 会发送验证邮件到你的邮箱，点击链接完成验证。

开始使用

注册完成后，你就可以在疑问区提问、在 Issue 中讨论了。

如何贡献资源

Fork 仓库

在 仓库页面 点击右上角的 Fork 按钮，复制仓库到你的账号。

编辑数据文件

在你的 Fork 中，编辑 _data/resources.yml 文件，按照已有格式添加新的资源条目。

提交 PR

修改完成后，点击"Create Pull Request"，描述你添加了什么资源。管理员审核通过后就会合并。
