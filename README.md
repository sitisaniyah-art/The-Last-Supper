# 最后的晚餐

AI协助搭建的资源互助社区

## 功能

- **学习资源库** — 高数、线代、编程、英语等多学科资源，按年级和类型分类
- **疑问区** — 提出学习问题，大家一起解答（基于 GitHub Issues）
- **社区贡献** — 上传资源、回答问题，贡献榜展示
- **暗黑模式** — 保护眼睛，适合深夜学习

## 技术栈

- [Jekyll](https://jekyllrb.com/) + [Minimal Mistakes](https://mmistakes.github.io/minimal-mistakes/) 主题
- GitHub Pages 托管
- GitHub Issues API 驱动问答系统

## 贡献资源

1. Fork 本仓库
2. 编辑 `_data/resources.yml`，添加资源信息
3. 提交 Pull Request

详见 [贡献指南](https://sitisaniyah-art.github.io/The-Last-Supper/contribute/)。

## 本地运行

```bash
bundle install
bundle exec jekyll serve
```
