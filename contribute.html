贡献资源

这个平台是一个AI协助搭建的资源互助社区，你的每一份贡献都能帮助到更多同学。你可以通过以下方式贡献：

资源上传向导

不知道怎么提交？让向导帮你选择正确的方式！

方式一：通过 Issue 提交（推荐）

无需 Fork 和命令行，直接在 GitHub 上填写表单即可提交。审核通过后会自动创建 PR 并由管理员合并。

提交学习资源 推荐好去处 推荐生存指南
查看资源目录结构

方式二：Fork & PR

Fork 本仓库

访问 GitHub 仓库，点击右上角的 Fork 按钮，将仓库复制到你的账号下。

添加资源数据

编辑 _data/resources.yml 文件，在末尾添加你的资源信息。格式如下： 

- id: 16
  title: 你的资源标题
  category: 数学与统计类       # 20个学科大类之一
  subcategory: 基础数学        # 对应大类下的具体课程
  grade: 大一                  # 大一/大二/大三/大四/通用
  type: 课件讲义               # 视频课程/课件讲义/书籍/习题试卷/其他
  uploader: 你的昵称
  date: 2026-05-15
  downloads: 0
  description: 资源的详细描述
  link: https://你的下载链接
  tags: [标签1, 标签2]

提交 Pull Request

修改完成后，提交 Pull Request。我们会尽快审核并合并你的贡献。
