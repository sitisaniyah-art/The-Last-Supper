常见问题

这里汇总了同学们最常问的问题，点击即可展开查看答案。

这个网站是做什么的？
"最后的晚餐"是一个学长学姐为学弟学妹搭建的学习资源互助平台。你可以在这里找到各科的学习资料（课件、真题、笔记等），也可以在疑问区提出学习问题，学长学姐会为你解答。
如何找到我需要的学习资源？
进入学习资源页面，你可以通过三种方式找到资源：
  • 按学科筛选：选择数学、编程、英语、物理等学科分类
  • 按年级筛选：选择大一、大二、大三或通用
  • 搜索关键词：在搜索框中输入资源名称或标签
如何提问？
进入疑问区，点击"我要提问"按钮，会跳转到 GitHub 的 Issue 页面。填写问题标题、分类和详细描述后提交即可。支持匿名提问，也可以直接粘贴截图。
如何贡献自己的学习资源？
请查看贡献资源页面，按照步骤 Fork 仓库、添加资源数据、提交 Pull Request 即可。贡献的资源会显示在贡献排行榜上。
贡献排行榜的积分是怎么算的？
  • 上传资源：+10 分
  • 回答问题：+5 分
  • 回答被采纳：+15 分
积分会定期更新，排行榜每月评选最佳贡献者。
下载链接失效了怎么办？
如果发现某个资源的下载链接失效，请在该资源对应的 GitHub Issue 中留言，或者直接联系我们。我们会尽快更新链接。
可以匿名提问吗？
可以。在提问表单中，"昵称"和"联系方式"都是选填项，留空即为匿名提问。但建议至少提供昵称，方便学长学姐称呼你。
我是新生，不太会用 GitHub，怎么办？
不用担心！请查看使用指南页面，里面有详细的新手教程。简单来说，你只需要一个 GitHub 账号就可以提问。贡献资源需要多几个步骤，但指南里都有图文说明。
如何联系管理员？
请查看关于我们页面，里面有管理员的联系方式。你也可以直接在 GitHub 仓库中提 Issue。
